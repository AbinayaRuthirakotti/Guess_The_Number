# Guess the Number Game

## Description
Guess the Number Game is a simple web-based game built using HTML, CSS, and JavaScript. The system generates a random number between 1 and 10, and the user tries to guess it. The game gives instant feedback such as Too High, Too Low, or Correct Guess, along with color-based visual indication and attempt counting.

---

## Features
- Random number generation (1–10)
- Real-time feedback
- Color-changing result indicator
- Attempt counter
- Reset game option
- Beginner-friendly interface

---

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Play
1. Enter a number between 1 and 10.
2. Click the **Check Guess** button.
3. View the result message and color indication.
4. Keep guessing until you win.
5. Click **Reset** to start again.
